# Agent Skills

This file contains documented procedures and specialized knowledge for frequent or complex tasks.

## PHP / Laravel
- **API Response:** Use `ApiResponse` trait for standardized JSON responses.
- **Validation:** Prefer `Request` validation within controllers or FormRequests.
- **RBAC:** Management of user-role and role-permission relationships.

## Workflow Procedures
- **Plan Mode:** Entering plan mode for all multi-step changes.
- **Codebase Analysis:** Systematic scanning of routes, models, and controllers.
